Carpaccio Pricing Spec & Testing

Input Parameters

  • price - the price of an item in the transaction
  • quantity - the number of items in the transaction
  • state - the state of residence of the customer for this transaction

Response

Content-Type

Response content type is expected to be application/json

Success (status code 200)

{
  price: number
}
As example: 
{
  price: 2390.99
}

Failure (all other status code)

{
  error: string
}
